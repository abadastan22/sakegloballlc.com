# SAKE Global LLC Website

Static website for `sakegloballlc.com`.

## Structure

- `index.html` - Homepage
- `pages/about.html` - About page
- `pages/services.html` - Services page
- `pages/government.html` - Government contracting page
- `pages/capabilities.html` - Capabilities page
- `pages/contact.html` - Contact page
- `assets/css/styles.css` - Main stylesheet
- `assets/js/main.js` - Mobile menu and dynamic year
- `assets/img/emboss-sg-logo.png` - Brand logo and favicon
- `assets/img/favicon.svg` - Original favicon mark

## Deployment

Upload the full folder contents to your hosting root/public directory. For GitHub Pages, commit all files and set the repository source to the root branch folder.

## Replace before going live

- Replace `contact@sakegloballlc.com` if needed.
- Add verified UEI, CAGE, certifications, insurance details, and past performance.
- Submit the contact form once after deployment and confirm the activation email sent to `contact@sakegloballlc.com`.
